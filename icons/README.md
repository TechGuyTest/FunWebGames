# App Icons

This directory should contain PNG icons for the PWA in the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Creating Icons

You can create icons using:
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Any image editor (export as PNG)

## Recommended Design

Use the game controller emoji 🎮 or create a colorful, child-friendly icon that represents FunWebGames.

**Note:** The PWA will work without custom icons, but will show a default icon until these are added.
